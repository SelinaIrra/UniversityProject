Основы программирования на языке Python

Python — это один из самых популярных и востребованных языков программирования. Он легок в изучении и универсален, поэтому хорошо подойдёт в качестве первого изучаемого языка для тех, кто никогда не занимался программированием.

Несмотря на доступность Python – это мощный инструмент для создания программ самого разнообразного назначения. Самые популярные направления применения языка Python: веб-разработка; data science: машинное обучение, анализ данных и визуализация; автоматизация процессов.

Популярность Python продолжает расти. Его используют крупные компании и стартапы. Например, Google, Yandex, Instagram, Intel, Positive Technologies, Eve Online, Houdini.

В ходе обучения школьники приобретают навыки программирования на языке Python: основы программирования, базовые алгоритмы и структуры данных, основы объектно-ориентированного программирования, работу с файлами и объектами операционной системы, работу с Интернет-ресурсами, создание законченных приложений.

На курсы принимаются школьники с 12 лет.

Занятия проводятся по будням 1 раз в неделю с 15:30 до 18:00.
